1. Inštalácia django do python:
    pip install django
    pip list

2. vytvorenie django projektu:
    django-admin startproject nazov_projektu .

3. nastavenie slovenčiny pre projekt (mysite/settings.py):
    LANGUAGE_CODE = 'sk'
    TIME_ZONE = 'Europe/Bratislava'

4. spustenie vývojového servera
    python manage.py runserver
    python.exe .\manage.py runserver

5. vytvorenie aplikácie
    python manage.py startapp nazov_aplikacie

6. zaregistrovanie aplikácie do projektu (mysite/settings.py):
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
        'nazov_aplikacie',
    ]

7. vytvorenie jednoduchého views (skuska/view.py)
from django.shortcuts import render, HttpResponse

def index(request):
    return HttpResponse('Hello World!')

8. vytvorenie urlpattern:
8.1 mysite/urls.py
    from django.contrib import admin
    from django.urls import path, include

    urlpatterns = [
        path('admin/', admin.site.urls),
        path('skuska/', include('skuska.urls')),
    ]
8.2
- vytvoriť súbor urls.py v priečinku aplikácie (skuska)
- upraviť urls.py:
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

----------------    9. TEMPLATE -------------------
9.1 
-   Create an app in folder templates/appName
-   in this folder create a html file (for ex. index.html)

9.2
-   edit views to use template (upraviť views tak aby po užíval template)

def index(request):
    return render(request, 'calculator/index.html')

10. CSS, obrázky (extérne)
- vytvorit v priecinku aplikacie priecinok static/mazov app
- v tomto priecinku vytvorit CSS (napr. calculator.css)
- do thot pricinku nakopirovat obrazky 
- do template (index.html) doplnit:
                                    - úplne na zaciatok {% load static %}
                                    - do hlavicky odkaz na style (calculator.css)
                                    <head>
                                        <link rel="stylesheet" href="{% static 'calculator/calculator.css' %}">
                                    </head>
- obrazok vlozit:
<img src={% static 'calculator/calculator.png' %}>
